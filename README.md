# 三国志演義 survivor

『三国志演義』全120回をたどるヴァンパイアサバイバー型ブラウザゲーム。
各回の主役を操作し、物量の敵を薙ぎ払いながらレベルアップ。その回の登場人物だけがレベルアップとガチャで集まる。遊ぶと演義120回の物語が頭に入る。

- ステージ＝120回。各回の主役を操作（前半ストーリー → 戦闘 → 後半ストーリー → ガチャ）。
- 武将の性能は「三国志演義 人物評価」（演義原文13軸採点）から決定論生成。武器種は各武将の逸話から（号令なし）。
- クリアで5連ガチャ、失敗で1連。排出はその回の登場武将のみ。
- 難易度3段（やさしい/ふつう/むずかしい）。
- Vanilla JS + Canvas、依存ライブラリなし。

遊ぶ: https://ngmt4amtk-web.github.io/sangokushi-engi-survivor/

姉妹作: 正史版『三國survivor』（武将＝武器を正史ランキングで決定論生成）。
