# 口調表(会話劇の憲法・主要人物)
台詞は一人2文以内・短く。地の文は最小限(場面転換と動作のみ)。口調は誇張しすぎない(時代劇の品格)。
- 劉備: 誠実で温かい。「〜だ」「〜してくれぬか」。民の話で熱くなる
- 関羽: 重厚・寡黙・古風。「〜である」「この関羽が」。義の言葉
- 張飛: 直情で声がでかい。「おい！」「〜だぜ」「兄者」。酒と喧嘩
- 諸葛亮: 静謐・理知。「〜です」「〜でしょう」。比喩で諭す
- 趙雲: 実直で清冽。「〜します」。主君への忠言は率直
- 馬超: 誇り高く激しい。「父の仇」が core
- 黄忠: 老いてなお血気。「老いぼれと侮るか」
- 魏延: ぶっきらぼうで野心的。「俺なら出来る」
- 姜維: 生真面目な熱血。「丞相」を慕う
- 龐統: 皮肉屋の天才。飄々と核心を突く
- 曹操: 不敵・断定・短文。「面白い」「殺せ」。詩心も
- 夏侯惇: 豪直。「応！」
- 司馬懿: 慇懃無礼。「これはこれは」。本心を見せない
- 郭嘉: 軽やかな天才。病んでも飄々
- 張遼: 折り目正しい武人
- 許褚: 単純豪快。「虎痴と呼べ」
- 典韋: 朴訥で忠義一途
- 孫堅: 豪放な父。「江東の虎よ」
- 孫策: 若く苛烈。「天下は駆けて取る」
- 孫権: 若き君主の慎重と決断。「皆の意見を聞こう」→決める時は鋭い
- 周瑜: 才気と矜持、諸葛亮への対抗心。「美周郎と呼ばれた男だ」
- 魯粛: 温厚な苦労人。板挟みでおろおろ→締めるとこは締める
- 呂蒙: 叩き上げの実直→学んでから自信
- 陸遜: 物腰柔らかいが切れ味鋭い。「書生とお笑いください」
- 甘寧: 蛮気と俠気。「鈴の音が聞こえたか」
- 呂布: 傲岸不遜。「この呂布に敵はおらぬ」。情に薄く欲に弱い
- 董卓: 下卑た尊大。「わしが天下よ」
- 貂蝉: 嫋やかだが芯が強い。敬語
- 王允: 老臣の悲壮な決意
- 袁紹: 貴族的な虚勢。決断が遅い
- 袁術: 増長と虚栄。「皇帝はわしだ」
- 張角: 神がかった predicador。「蒼天已死…」
- 司徒・文官(汎用): 「申し上げます」「恐れながら」
- 兵士(汎用): 「はっ！」「て、敵襲ーー！」
- 老人・民(汎用): 素朴。「ありがたや」
- ナレーション(地の文): 現行ナレーション様式を簡潔化。1行で場面と動作だけ。感情は台詞に譲る
